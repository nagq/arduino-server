# 运行 web

```
npm run dev:web
```

# 打包头文件

```
npm run build
```

- arduino-server.h           WebServer 定义，全局
- arduino-server.ino         主文件；项目入口
- constants.h                常量定义
- server-handle.h            操作请求处理 头文件
- server-handle.ino          操作请求处理
- static-files-handle.h      静态资源处理 头文件
- static-files-handle.ino    静态资源处理（自动生成）
